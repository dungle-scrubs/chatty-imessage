## [0.1.0] - 2026-02-15

### Added

- List messages with date range, contact, unread, and attachment filters
- Natural language date expressions (`--last "15 days"`, `--this-week`, etc.)
- Send messages via iMessage or SMS to phone numbers, emails, or contact names
- Contact name resolution from Contacts.app
- Open message attachments directly from CLI
- JSON output for scripting
- Verbose mode with delivery times and expressive effects
