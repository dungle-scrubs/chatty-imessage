# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a
[private security advisory](https://github.com/dungle-scrubs/chat-imessage/security/advisories/new)
on GitHub.

Do **not** open a public issue for security vulnerabilities.

You should receive an initial response within 72 hours. If the vulnerability is
confirmed, a fix will be released as soon as possible.
